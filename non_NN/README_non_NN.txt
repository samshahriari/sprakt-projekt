Make sure to have en.vectors downloaded and in the data folder. 
Make sure to have at least train.csv and test.csv in the data folder.

Run run_first_time.sh This will save files for w2id, position cache, vector cache allowing
for much faster execution the second time. This only has to be run 1 time. 

Run run_with_cache.sh
There are 6 different configurations to try. All but one of these is commented out in the run_with_cache.sh file.
